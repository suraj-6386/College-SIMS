Role-Based Directory Structure:

/pages/admin/
- admin-dashboard.jsp - Admin main entry point with quick stats.
- admin-pending.jsp - Approve or reject new student/teacher registrations.
- admin-users.jsp - Manage all active users in the system.
- courses.jsp - Add or update courses and subjects.
- subjects.jsp - Subject management and teacher assignment.
- reports.jsp - System-wide academic and enrollment analytics.
- get-user-details.jsp - AJAX helper for fetching user metadata.

/pages/teacher/
- teacher-dashboard.jsp - Teacher's primary overview.
- teacher-attendance.jsp - Mark and edit student attendance.
- teacher-marks.jsp - Enter and update student grades.
- teacher-students.jsp - View students enrolled in assigned subjects.
- teacher-courses.jsp - View all assigned subjects.
- teacher-profile.jsp - Personal faculty profile.
- get-subject-students.jsp - AJAX helper for loading students by subject.
- get-subject-students-json.jsp - JSON API for student enrollment data.

/pages/student/
- student-dashboard.jsp - Student's academic overview.
- student-attendance.jsp - Personal attendance history.
- student-marks.jsp - View grades and marks.
- student-courses.jsp - List of enrolled subjects.
- student-profile.jsp - Personal academic profile.

/pages/common/
- login.jsp - Unified login portal for all roles.
- registration.jsp - Shared registration form for students and teachers.
- logout.jsp - Session termination and redirect to index.
- announcements.jsp - Global announcement feed.
- error.jsp - Universal error handling page.

/configure/
- DBConnection.jsp - Centralized database connection logic.

/styles/
- style.css - Global "Digital Craftsmanship" design system.

/images/
- logo.png - Main system logo asset.
- image.png - Background or auxiliary images.

Root:
- index.html - Landing page with login and registration entry points.
